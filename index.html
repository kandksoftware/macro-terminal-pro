About CNC Macro Simulator II

Due to the complexity of CNC macro programming (also known as CNC parametric programming) and better understanding them, we have created an application well known as Macro Terminal CNC available on the App Store and Google Play.
Available in both milling and turning versions, this application quickly gained great success and great reviews, especially in the US market.
With the ability to use system variables, a variety of subroutine formats, and an innovative cnc macro analysis tool that allows you to see all logical and arithmetic evaluations, it was second to none.
This success and many inquiries about the desktop version encouraged us to create a new, lightweight and much more powerful cnc programmer's tool which is CNC Macro Simulator.
Currently, CNC Macro Simulator not only runs on desktop and mobile devices, but can also run on a variety of operating systems.
Since the end of 2022, CNC Macro Simulator is equipped with a conversational shell, thanks to which creating a CNC program does not even require manual code writing.
Centroid control support has also been added since early 2023, which, combined with already existing support for Fanuc Custom Macro B and Haas, makes CNC Macro Simulator the most versail CNC macro programming and CNC programming software on the market.
