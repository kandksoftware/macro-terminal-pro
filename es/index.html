¿Qué es CNC Macro Simulator II?

CNC Macro Simulator II es un simulador y editor de código CNC liviano diseñado para agilizar la creación de programas macro para controles como Fanuc, Haas, Mazak (ISO), Centroid e incluso Sinumerik, Heidenhain u Okuma con convertidores integrados. Admite fresadoras y tornos de 3 ejes con eje C/Y y herramientas en vivo. El software funciona sin conexión en Windows, Linux, macOS, Android e iOS, y está disponible para computadoras de escritorio, tabletas y teléfonos móviles. Si bien el simulador actualmente representa solo 3 ejes, el software puede ejecutar programas macro para hasta 6 ejes, lo que ofrece depuración y conversión a código G simple.
